Projeto integrado na Unidade Currirular de Sistemas Distrbuídos

Contribuídores: 24Preacher, A82516, pedromendespinto

Para utilizar corretamente a aplicação é necessário alterar os paths nas classes Cloud e ServidorReader do Package Servidor e na classe UtilizadorReader do Package Utilizador.
Não é possível utilizar espaços, por isso aconselhamos a utilizar underscore.
Para dar Upload a uma música, esta deverá estar dentro do Package Upload.
O Download de uma música será realiszado para o Package Download.
